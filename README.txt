StudelyLedger - Projet complet starter

Installation
1. Dézipper dans htdocs.
2. Créer une base et importer database.sql via phpMyAdmin.
3. Vérifier APP_BASE_PATH dans config/app.php.
4. Ouvrir /studelyledger_full_project/login.php

Compte démo
- admin / password

Ce pack branche :
- login réel
- dashboard réel
- clients
- imports CSV avec preview
- journal des imports
- correction des lignes rejetées
- action manuelle
- frais en masse

Pour gérer Excel .xlsx, ajoute PhpSpreadsheet dans le projet.
